### Prerequisites

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/nandaydas/Portfolio.git
   cd Portfolio
   ```

2. **Open the project**

   ```bash
   # Open with VS Code
   code .

   # Or open index.html directly in your browser
   open index.html
   ```

3. **Local Development**

   ```bash
   # Using Python (if available)
   python -m http.server 8000

   # Using Node.js (if available)
   npx http-server

   # Or use VS Code Live Server extension
   ```

4. **View the website**
   - Open `http://localhost:8000` in your browser
   - Or right-click `index.html` and open with your preferred browser
