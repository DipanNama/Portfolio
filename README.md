## 🎯 About

A modern, responsive, and interactive portfolio website showcasing my skills, projects, and experience as a **Frontend Developer** and **Cybersecurity Enthusiast**. Built with vanilla JavaScript, HTML5, and CSS3, featuring smooth animations, dark/light theme toggle, and a fully functional contact form.

### ✨ Key Highlights

- 🎨 **Modern Design**: Clean, professional interface with smooth animations
- 📱 **Fully Responsive**: Optimized for all devices and screen sizes
- 🌙 **Theme Toggle**: Dark/Light mode with smooth transitions
- 🛡️ **Security Focus**: Cybersecurity projects and CTF achievements
- 💼 **Professional**: Resume download, project showcases, and contact form
- ⚡ **Performance**: Fast loading with optimized assets and animations

## 🚀 Getting Started

### Prerequisites

- Web browser (Chrome, Firefox, Safari, Edge)
- Code editor (VS Code recommended)
- Git (for cloning)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/nandaydas/Portfolio.git
   cd Portfolio
   ```

2. **Open the project**

   ```bash
   # Open with VS Code
   code .

   # Or open index.html directly in your browser
   open index.html
   ```

3. **Local Development**

   ```bash
   # Using Python (if available)
   python -m http.server 8000

   # Using Node.js (if available)
   npx http-server

   # Or use VS Code Live Server extension
   ```

4. **View the website**
   - Open `http://localhost:8000` in your browser
   - Or right-click `index.html` and open with your preferred browser

## 🎨 Customization

### **Personal Information**

1. Update personal details in `index.html`:
   - Name, title, and bio
   - Contact information
   - Social media links
   - Resume link

### **Projects**

1. Add your projects in the projects section:
   - Project images in `assets/`
   - Update project cards with your details
   - Add live demo and GitHub links

### **Skills**

1. Modify the skills section:
   - Add/remove technology icons
   - Update skill categories
   - Adjust proficiency levels

### **Styling**

1. Customize colors in `css/style.css`:
   ```css
   :root {
     --accent-primary: #14ae96; /* Primary accent color */
     --accent-secondary: #8b5cf6; /* Secondary accent color */
     /* Add your custom colors */
   }
   ```

### **Content**

1. Replace placeholder content:
   - Update project descriptions
   - Add your achievements
   - Modify service offerings
